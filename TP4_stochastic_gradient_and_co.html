II. The optimization problem

Logistic regression consists in finding the parameter $x \in \mathbb{R}^d$ that best fits: $$ b_i = \begin{cases} \phantom{-}1 \ \textrm{ if }\ \langle a_{i}, x\rangle \ge 0 \\ -1 \ \textrm{ if }\ \langle a_{i}, x\rangle < 0 \end{cases} $$ for all the samples $i$ of the training set (where $a_i$ is the $i^{th}$ line of $A$ and $b_i$ the $i^{th}$ element of $b$). To measure this fit, logistic regression chooses the following loss:

$$ \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) = \log \left(1+\exp \left(-b_{i}\left\langle a_{i}, x\right\rangle\right)\right) $$

The corresponding optimization problem we will consider is the following:

Minimization of the empirical risk $f$ of the $L_2$-regularized logistic regression model, on the dataset $(A_{\textrm{train}},b_{\textrm{train}})$: $$ \begin{aligned} \textrm{min}_{x\in \mathbb{R}^d} \ f(x) \ &= \ \textrm{min}_{x\in \mathbb{R}^d} \ \frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} f_i(x) \\ &= \ \textrm{min}_{x\in \mathbb{R}^d}\frac{1}{n_{\textrm{train}}} \sum_{i=1}^{n_{\textrm{train}}} \textrm{LogisticLoss}(b_i, \left\langle a_{i}, x\right\rangle ) + \frac{l_2}{2}\|x\|^{2}_2, \end{aligned} $$ where $l_2 \ge 0$ is the regularization parameter.

To simplify the notation, we'll stop indexing $A$, $b$ and $n$ with "$\textrm{train}$" in the following.

Note that the functions $f_i$s and $f$ are $C^1$ with $L$-Lischitz gradient, with $L \le \frac{\max_i\|a_i\|^2_2}{4} + l_2$ and we can therefore use the following value in our implementation:

L = 0.25 * max(np.linalg.norm(A,2,axis=1))**2 + self.l2

GD (Gradient descent)¶

At iteration k of the GD algorithm, the following update is executed:

$$ x_{k+1} \leftarrow x_{k} - \gamma\nabla f(x_k) $$

def GD(x0, grad, max_iter, n, L, mu):
    '''
    Parameters
    ----------
    x0: array, shape (nb_features,)
        Initialisation of the solver
    grad: function
        Gradient of the objective function
    max_iter: int
        Number of iterations (i.e. number of descent steps). Note that for GD, 
        one iteration is one epoch i.e, one pass through the data, while for SGD, SAG and SAGA, 
        one iteration uses only one data point.
    n: int
        Dataset size
    L: float
        Smoothness constant of the objective function
    mu: float
        Strong convexity constant of the objective function

    Returns
    -------
    x: array, shape (nb_features,)
        final iterate of the solver
    x_tab: array, shape (nb_features, max_iter)
        table of all the iterates
    '''
    if L == 0:
        return None, None

    stepsize  = 2/L ######## TODO (2) ########
    x         = x0
    x_tab     = np.copy(x)

    for k in range(max_iter):
        x     = x - stepsize* grad(x) ######## TODO (3): Implement a single step ########
        x_tab = np.vstack((x_tab, x))

    return x, x_tab

SGD (Stochastic Gradient Descent)¶

At iteration k of the SGD algorithm, the following update is executed:

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow x_{k} - \gamma_k \nabla f_i(x_k) \\ \end{cases} $$
Question:
  • What stepsize $\gamma_k$ ensures the convergence of this algorithm and under what assumptions?
  • Complete TODO (4)
  • Add your code to \lib\solvers.py.

Un pas constant égale à $\gamma_k = \gamma = \frac{\gamma_0}{(+\frac{\kappa}{\kappa_0})^{\alpha}}$. Avec $\alpha = 0,5$ ou $0,6$.

Pour le TP on prendre $\kappa_0 =1$ et $\gamma_0 = \frac{2}{L}$

import random as rd
def SGD(x0, grad, max_iter, n, L, mu):
    ######## TODO (4) ########
    x_tab = x0
    gamma0 = 2/L
    kappa = mu/L
    stepsize = gamma0/(1+kappa)**0.5
    for k in range(max_iter):
        ######## TODO (4) ########
        i = rd.randint(0,n-1)


        x = x - stepsize * grad(x,i)
        if k%n == 0: # each completed epoch
            x_tab = np.vstack((x_tab, x))

    return x, x_tab

SAGA (Stochastic Average Gradient "Amélioré")¶

SAGA starts by computing a single batch gradient, then at iteration k, the following update is executed:

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow x_{k} - \gamma_k\big(\nabla f_i(x_k) - \alpha_i + \bar{\alpha}\big) \\ \alpha_i \leftarrow \nabla f_i(x) \end{cases} $$

where $(\alpha_j)_{j=1}^n$ are gradients evaluated at previous iterates and $\bar{\alpha}=\frac{1}{n}\sum^n_{j=1}\alpha_j$.

Question:
  • What stepsize $\gamma_k$ ensures the convergence of this algorithm and under what assumptions?
  • Complete TODO (5)
  • Add your code to \lib\solvers.py.

En supposant que la fonction objective soit $\mu$-fortement convexe et $L$-fortement différentiable. On peut prendre $\gamma = \frac{1}{2(\mu n+L)}$

def SAGA(x0, grad, max_iter, n, L, mu):
    x_tab = x0
    x=x0
    alpha = [grad(x0, i) for i in range(0,n)]
    stepsize = 1/2/(mu*n+L)
    for k in range(max_iter):
        ######## TODO (5) ########
        i = rd.randint(0,n-1)
        alphab = sum(alpha)/n
        xk = np.copy(x)
        x = x - stepsize*(grad(x,i)-alpha[i]+alphab)
        alpha[i] = grad(xk,i)
        if k%n == 0: # each completed epoch
            x_tab = np.vstack((x_tab, x))

    return x, x_tab

Answer:

Pour le GD : $$x_{k+1} \leftarrow \textrm{prox}_{l_1 \|\cdot\|_1}(x_{k} - \gamma\nabla f(x_k))$$

Pour le SGD : $$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow \textrm{prox}_{l_1 \|\cdot\|_1}(x_{k} - \gamma_k \nabla f_i(x_k)) \\ \end{cases} $$

$$ \begin{cases} i \textrm{ chosen randomly in } \{1...n\}\\ x_{k+1} \leftarrow \textrm{prox}_{l_1 \|\cdot\|_1}(x_{k} - \gamma_k\big(\nabla f_i(x_k) - \alpha_i + \bar{\alpha}\big)) \\ \alpha_i \leftarrow \nabla f_i(x) \end{cases} $$

Answer:

$$\forall i \in [|1,d|]$$$$ (\textrm{prox}_{l_1 \|\cdot\|_1})_{i} \begin{cases} x_i - l_1 \ \ si \ \ x_1 \gt l_1 \\ x_i + l_1 \ \ si \ \ x_1 \lt l_1 \\ 0\ \ sinon \end{cases} $$ 
def prox(x,stepsize):
    ######## TODO (7) ########
    p = np.zeros(len(x))
    gamma = stepsize * self.l1

    for i in range(len(x)):
        if x[i] > gamma:
            p[i] = x[i] - gamma
        elif x[i] < -gamma:
            p[i] = x[i] + gamma
        else:
            p[i] = 0

    return p
